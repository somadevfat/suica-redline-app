# Suica PDF 赤線出力アプリ

モバイルSuicaの残高ご利用明細PDFを読み込み、明細行ごとに赤線を引いたPDFを個別出力するブラウザアプリです。

## 使い方

1. `index.html` をブラウザで開く
2. Suica明細PDFを選択
3. 検出された明細を確認
4. 「このPDFを出力」または「すべて個別PDFで出力」を押す

## 出力ファイル名

以下の形式で出力されます。

```txt
YYYYMMDD_入駅_to_出駅_redline.pdf
```

例:

```txt
20260612_渋谷_to_新宿_redline.pdf
```

## 使用ライブラリ

CDNで以下を読み込んでいます。

- pdf.js: PDF表示・テキスト位置取得
- pdf-lib: PDFへの赤線描画・出力

## テスト

アプリの利用にNode.jsは不要です。`index.html` をブラウザで開けば動きます。

文字列解析などの純粋関数だけ、Node.js標準のテストランナーで確認できます。

```sh
node --test suica.test.js
```

## 調整ポイント

赤線が少し上下にズレる場合は、`app.js` の以下を調整してください。

```js
y: y - 2,
```

数値を `y - 1` や `y - 3` に変えると赤線位置が変わります。
